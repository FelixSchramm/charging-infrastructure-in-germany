## Neues Modul `01_app/sections/data_table.py`

### Spaltenauswahl (eine Quelle für Tabelle **und** CSV-Export)

Eine kuratierte, umbenannte Teilmenge bauen (keine zwei getrennten Transformationen
pflegen — Tabelle und Download nutzen denselben DataFrame):

| Quellspalte | Anzeige-Label |
|---|---|
| `BetreiberBereinigt` | Betreiber |
| `Bundesland` | Bundesland |
| `KreisKreisfreieStadt` | Landkreis/Stadt |
| `Ort` | Ort |
| `PLZ` | PLZ |
| `Strasse` + `Hausnummer` (verkettet) | Adresse |
| `Inbetriebnahmedatum` | Inbetriebnahme |
| `LadeleistungInKW` | Ladeleistung (kW) |
| `Leistungskategorie` | Typ |
| `Steckertyp` | Steckertyp |
| `ArtLadeeinrichtung` | Art der Ladeeinrichtung |
| `Breitengrad`, `Laengengrad` | Breitengrad, Längengrad |

Breiten-/Längengrad bewusst mit aufnehmen (auch wenn in der Bildschirm-Vorschau
weniger relevant): Ein Datendownload wird oft für externe Weiterverarbeitung
(GIS, eigene Karten) genutzt, dafür sind die Koordinaten der Mehrwert.

```python
def _build_export_df(df_filtered: pd.DataFrame) -> pd.DataFrame:
    export = pd.DataFrame({
        "Betreiber": df_filtered["BetreiberBereinigt"],
        "Bundesland": df_filtered["Bundesland"],
        "Landkreis/Stadt": df_filtered["KreisKreisfreieStadt"],
        "Ort": df_filtered["Ort"],
        "PLZ": df_filtered["PLZ"],
        "Adresse": df_filtered["Strasse"].fillna("") + " " + df_filtered["Hausnummer"].fillna(""),
        "Inbetriebnahme": df_filtered["Inbetriebnahmedatum"].dt.strftime("%d.%m.%Y"),
        "Ladeleistung (kW)": df_filtered["LadeleistungInKW"],
        "Typ": df_filtered["Leistungskategorie"],
        "Steckertyp": df_filtered["Steckertyp"],
        "Art der Ladeeinrichtung": df_filtered["ArtLadeeinrichtung"],
        "Breitengrad": df_filtered["Breitengrad"],
        "Laengengrad": df_filtered["Laengengrad"],
    })
    return export
```

### Tabellen-Vorschau mit Zeilenlimit

Bei bis zu 201.100 Zeilen (ungefiltert) ist eine vollständige Übertragung ans
Frontend unnötig langsam. Vorschau auf z. B. die ersten 1.000 Zeilen begrenzen,
Hinweistext mit Gesamtzahl der gefilterten Zeilen ergänzen. Der **Download**
enthält immer den vollständigen gefilterten Datensatz, unabhängig vom Limit.

```python
_PREVIEW_ROWS = 1000

def render_data_table(df_filtered: pd.DataFrame):
    st.header("Datentabelle")

    export_df = _build_export_df(df_filtered)
    n_total = len(export_df)

    if n_total > _PREVIEW_ROWS:
        st.caption(
            f"Zeigt die ersten {_PREVIEW_ROWS:,} von {n_total:,} gefilterten "
            f"Zeilen (Punkt als Tausendertrennzeichen). Der Download enthält "
            f"alle gefilterten Zeilen."
        )
    else:
        st.caption(f"{n_total:,} gefilterte Zeilen.".replace(",", "."))

    st.dataframe(export_df.head(_PREVIEW_ROWS), use_container_width=True, hide_index=True)

    st.download_button(
        label="Als CSV herunterladen",
        data=_to_csv_bytes(export_df),
        file_name=f"ladeinfrastruktur_gefiltert_{pd.Timestamp.now():%Y-%m-%d}.csv",
        mime="text/csv",
    )
```

> Zahlenformat im Caption-Text: Python `f"{n:,}"` erzeugt Komma als
> Tausendertrenner — analog zu `_de()` in `kpis.py` mit `.replace(",", ".")`
> auf deutsches Format umstellen (nicht einfach `:,` stehen lassen).

### CSV-Erzeugung: deutsche Excel-Konventionen beachten

Deutsches Excel erwartet Semikolon als Trennzeichen und Komma als
Dezimaltrennzeichen; `utf-8-sig` (mit BOM) sorgt dafür, dass Umlaute in Excel
korrekt angezeigt werden (reines `utf-8` zeigt in Excel oft kaputte Umlaute):

```python
def _to_csv_bytes(export_df: pd.DataFrame) -> bytes:
    """CSV mit deutschen Excel-Konventionen (Semikolon, Dezimalkomma, BOM)."""
    return export_df.to_csv(index=False, sep=";", decimal=",").encode("utf-8-sig")
```

Optional (nice-to-have, kein Muss): `_to_csv_bytes` mit `@st.cache_data`
versehen, damit die CSV-Konvertierung bei unveränderten Filtern nicht bei
jedem Rerun (z. B. durch Interaktion in einem anderen Tab) neu läuft. Cache-Key
ist der DataFrame-Inhalt selbst (`st.cache_data` hasht Pandas-DataFrames nativ).
